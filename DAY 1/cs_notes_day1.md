HTTP: The basic language of the web. A protocol used by browsers and apps to send requests and get responses over the web, using methods like GET, POST, PUT, DELETE.

Client: the app that sends a request (browser, mobile app, your Python code).
Server: the machine that receives the request, processes it, and sends back data (like JSON/HTML) with a status code.

REST is a style of designing APIs that use HTTP in a clean, predictable way.

REST API: An API that uses HTTP in a structured way: resources identified by URLs (like /users/1) and standard methods (GET = read, POST = create, PUT = update, DELETE = delete). [CRUD]
